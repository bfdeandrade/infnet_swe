Questão 13 TP2 - HTML/CSS

1)Seletor de Tag

Seleciona todos os componentes são da mesma tag

2)Seletor de Classe

Seleciona todos os itens que possuem a mesma classe. Para utilizar-la, utiliza o . + nome da classe

3)Seletor de ID

Seleciona todos os itens que possuem o mesmo id="id"

4)Seletor de Atributo

Seleciona os itens com base no atributo

5)Seletor de Descendente

Aplica a todos as tags que estão dentro de uma tag mãe

6)Seletor universal

Você pode usar um seletor universal '*' para aplicar uma regra para todos os componentes de uma página HTML
